# FoodPlease+ - Aplicación de Pedidos de Comida

FoodPlease+ es una versión mínimamente viable (MVP) de una aplicación de pedidos de comida con tres tipos de usuario: Cliente, Dueño del Local y Repartidor (Delivery).

## Requisitos
- Navegador web (Chrome, Edge, Firefox).

## Pasos para ejecutar el proyecto

1. Descargar el repositorio desde GitHub.
2. Abrir la carpeta del proyecto.
3. Abrir el archivo index.html.
4. La aplicación se ejecutará automáticamente en el navegador.

## Funcionalidades

### Cliente
- Ver productos
- Agregar al carrito
- Confirmar pedido
- Ver estado del pedido

### Dueño del local
- Agregar productos
- Editar productos
- Eliminar productos

### Repartidor (Delivery)
- Ver pedidos asignados
- Cambiar estado del pedido

## Tecnologías utilizadas
- HTML
- CSS
